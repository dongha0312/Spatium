Before you start

좋은 사진이 좋은 모델을 만들어요

아래 세 가지만 확인하면 배경 제거와 3D 생성 결과가 훨씬 안정적입니다.

☀

밝고 고른 조명

강한 그림자나 역광을 피하고 피규어의 색과 질감이 선명하게 보이도록 촬영하세요.

피규어와 비슷한 색의 물건이 겹치지 않는 사진일수록 객체를 더 정확히 찾을 수 있어요.

01

서랍장에 꾸밀 사진을 업로드 해요

서랍장 꾸미기 화면에서 사진을 끌어다 놓거나 업로드 영역을 눌러 파일을 선택하세요.

  • JPG, PNG, WEBP 이미지 파일을 사용할 수 있어요.
  • 미리보기에서 피규어 전체가 선명한지 확인하세요.
  • 바꾸려면 ‘다른 사진 선택’을 누르면 됩니다.
TIP서랍장에 올릴 사진이 함께 찍혔다면 만들고 싶은 피규어가 다른 물체에 가려지지 않은 사진을 선택하세요.
02

객체명을 입력하고 모델을 선택해요

사진에서 분리할 피규어 이름을 입력한 뒤, 배경 제거와 3D 생성에 사용할 모델과 품질 설정을 선택하세요.

  • 사진에서 찾기 쉬운 이름을 입력하세요. 예: 곰인형, 로봇 피규어, 미니어처 자동차.
  • 객체 분리 모델을 선택해 입력한 이름을 기준으로 피규어를 찾게 하세요.
  • 3D 생성 모델, 텍스처 해상도와 리메시 방식을 확인한 후 다음으로 이동하세요.
TIP객체명은 배경 제거 대상을 찾는 기준입니다. 사진 속 피규어를 가장 잘 설명하는 이름을 입력하세요.
03

배경 제거 결과를 확인해요

입력한 객체명을 기준으로 찾은 피규어만 남고 주변은 체크무늬로 표시됩니다. 분리 결과와 탐지 정보를 확인하세요.

  • ‘원본 사진 보기’로 제거 전·후를 비교할 수 있어요.
  • 분리 결과의 객체명과 탐지 신뢰도가 의도한 피규어와 맞는지 확인하세요.
  • 피규어 일부가 사라졌다면 이전 단계에서 객체명을 고쳐 다시 진행하세요.
TIP피규어의 가느다란 팔다리처럼 얇은 부분이 잘 보존됐는지가 3D 모델의 완성도에 큰 영향을 줍니다.
04

3D 모델 생성을 기다려요

배경이 제거된 이미지를 분석해 형상과 텍스처를 가진 GLB 모델을 만듭니다. 진행률이 100%가 될 때까지 기다려주세요.

  • 이미지 전처리 → 형상 복원 → 텍스처 생성 순서로 진행됩니다.
  • 처리 중에는 브라우저 창을 닫거나 새로고침하지 마세요.
  • 완료되면 ‘다음’ 버튼이 활성화됩니다.

05

모델 방향과 크기를 보정해요

생성 직후 모델이 눕거나 방향이 어긋날 수 있어요. 3D 뷰어에서 실제 피규어처럼 서도록 회전과 위치를 보정하세요.

  • 먼저 ‘자동 정렬’을 눌러 기본 방향을 맞춰보세요.
  • X·Y·Z축 ±90° 버튼과 회전 핸들로 방향을 세밀하게 조정하세요.
  • 피규어가 바닥에 자연스럽게 닿으면 ‘보정 완료 · 저장하기’를 누르세요.

06

이름과 카테고리를 확인하고 저장해요

마지막으로 피규어 이름과 분류, 생성된 파일 정보를 확인한 뒤 내 서랍장에 추가하세요.

  • 에디터에서 찾기 쉬운 이름으로 수정할 수 있어요.
  • 피규어, 인형, 미니어처, 소품 등 알맞은 카테고리를 선택하세요.
  • 저장이 끝나면 3D 에디터에서 바로 배치할 수 있어요.
DONE저장 완료 메시지가 보이면 모든 과정이 끝났습니다. 이제 3D 에디터의 내 서랍장에서 사용할 수 있어요.
Troubleshooting

결과가 마음에 들지 않을 때

피규어를 찾지 못해요

피규어 전체가 보이고 배경과 색이 잘 구분되는 사진으로 바꿔보세요. 객체명도 ‘피규어’가 아닌 ‘테디베어 인형’처럼 구체적으로 입력하면 좋습니다.

배경이 일부 남았어요

입력한 객체명이 사진 속 피규어를 정확히 설명하는지 확인하세요. 이름을 더 구체적으로 고치거나 다른 객체 분리 모델을 선택한 뒤 다시 진행해보세요.

모델 방향이 뒤집혔어요

모델 보정 화면의 X·Y·Z 축 회전 버튼을 90도 단위로 먼저 맞춘 다음, 주황색 회전 핸들로 세밀하게 조정하세요.
