Before you start

스캔하기 좋은 방을 준비하세요

아래 세 가지만 확인하면 벽과 가구의 위치를 더 안정적으로 기록할 수 있습니다.

▣

프로젝트를 먼저 준비

첫 스캔을 시작하면 새 프로젝트 생성 화면이 열립니다. 공간을 묶어 관리할 프로젝트 이름을 정한 뒤 스캔으로 이동하세요.

☀

조명과 이동 경로 확보

커튼을 열거나 조명을 켜고, 방 가장자리를 천천히 돌 수 있도록 바닥의 장애물을 치워주세요. 거울이나 단색 벽만 오래 비추지 않는 것이 좋습니다.

⌁

지원 기기와 카메라 권한

Apple RoomPlan을 사용할 수 있는 iPhone 또는 iPad에서 진행하고, SPATIUM의 카메라 접근을 허용하세요. 배터리도 충분히 충전해두세요.

01

스캔 탭에서 프로젝트를 준비해요

하단 메뉴의 스캔을 열고 방 스캔 시작을 누르세요. 첫 스캔이라면 새 프로젝트를 만든 뒤 카메라 화면으로 이동합니다.

  • 새 공간은 프로젝트를 만든 뒤 스캔이 자동으로 시작됩니다.
  • 기존 프로젝트에 방을 추가할 때는 해당 프로젝트의 방 추가 기능을 이용하세요.
  • 이전 스캔이 있다면 우측의 다시 스캔 버튼으로 새 스캔을 시작할 수 있어요.
TIP 프로젝트는 여러 방을 한곳에 모아 관리하는 단위입니다. 알아보기 쉬운 집·공간 이름을 사용하세요.
02

기기를 움직여 스캔을 준비해요

카메라가 열리면 화면의 안내에 따라 기기를 천천히 좌우로 움직이세요. RoomPlan이 주변 공간을 인식하면 스캔 가능한 상태가 됩니다.

  • 기기를 세로로 잡고 급하게 흔들지 마세요.
  • 바닥과 벽의 경계가 화면에 함께 보이도록 시작하면 좋아요.
  • 카메라 권한 안내가 나오면 허용해야 스캔할 수 있습니다.
TIP 인식이 시작되지 않으면 밝은 벽 모서리나 바닥처럼 형태가 뚜렷한 곳을 천천히 비춰보세요.
03

방을 한 바퀴 천천히 비춰주세요

한쪽 벽에서 시작해 같은 방향으로 이동하면서 벽·바닥·가구·문·창문이 모두 보이게 비추세요. 인식된 면과 객체는 흰 선 및 하단의 미니 3D 모델로 표시됩니다.

  • 벽의 위·아래와 방 모서리를 빠뜨리지 말고 연결해 주세요.
  • 가구 앞에서 잠시 속도를 늦춰 형태가 충분히 잡히도록 하세요.
  • 왼쪽 카메라 버튼을 누르면 결과와 함께 보관할 참고 사진을 추가할 수 있어요.
중요 카메라 버튼은 참고 사진을 저장하는 기능입니다. 누르지 않아도 RoomPlan의 3D 공간 스캔은 계속됩니다.
04

빠진 곳을 확인하고 스캔을 완료해요

시작 지점까지 한 바퀴 돌았다면 하단 미니 모델에서 방 윤곽을 확인하세요. 주요 면과 가구가 충분히 보이면 스캔 완료를 누르고 결과 처리가 끝날 때까지 기다립니다.

  • 벽이나 바닥에 큰 빈틈이 보이면 그 방향을 다시 천천히 비춰주세요.
  • 스캔 완료를 누른 뒤에는 기하 데이터 처리 화면으로 전환됩니다.
  • 처리가 끝나면 스캔 탭의 결과 검토 화면으로 자동 이동합니다.
TIP 처리 중 앱을 닫지 마세요. 오류가 발생하면 스캔 탭으로 돌아온 뒤 조명을 밝게 하고 다시 시도하세요.
05

방 정보와 감지된 요소를 검토해요

결과 화면에서 방 이름, 감지된 객체의 종류·크기, 촬영 사진 수를 확인하세요. 방 정보에 입력한 값은 함께 저장되는 metadata JSON의 roomType으로 전달됩니다.

  • ‘이름 없는 공간’ 대신 침실·거실·주방처럼 구체적인 이름을 입력하세요.
  • 감지 요소 목록에서 객체 이름과 가로 × 높이 × 깊이 치수를 확인하세요.
  • 촬영 사진이 0장이어도 3D 스캔 결과는 저장할 수 있습니다.
TIP 객체가 빠졌거나 잘못 인식되어도 3D 편집 화면에서 이동·수정·추가·삭제할 수 있습니다.
06

목적에 맞게 편집하거나 저장해요

결과를 다듬으려면 3D로 편집하기, 파일로 보관하려면 파일 공유, 현재 프로젝트에 저장하려면 서버로 업로드를 선택하세요.

  • 3D 편집에서는 객체를 이동·수정·추가·삭제하고 저장할 수 있어요.
  • 파일 공유는 USDZ, metadata JSON, 촬영한 JPG 사진을 묶어 내보냅니다.
  • 서버 업로드가 끝나면 프로젝트의 방 목록으로 이동합니다.
USDZ
room-scan.usdz스캔된 방의 3D 메시
JSON
ai-edit-request.json방 이름과 감지·편집 객체 정보
중요 이미 3D 편집에서 서버 저장을 마친 스캔은 중복 방 생성을 막기 위해 다시 업로드되지 않습니다.
Troubleshooting

스캔이 잘 되지 않을 때

스캔 준비 중에서 넘어가지 않아요

조명을 밝게 켜고 형태가 분명한 벽 모서리와 바닥을 비추며 기기를 천천히 좌우로 움직이세요.

벽이나 가구가 빠졌어요

빠진 면과 물체를 다른 각도에서 다시 비춰주세요. 완료 후 객체 문제는 3D 편집에서도 보정할 수 있습니다.

방 사진이 0장으로 보여요

사진은 자동 촬영되지 않습니다. 스캔 중 왼쪽 카메라 버튼을 눌렀을 때만 참고 사진이 추가됩니다.

스캔을 다시 하고 싶어요

결과 화면 상단의 다시 스캔 버튼을 누르세요. 현재 검토 중인 스캔을 비우고 새 스캔을 시작합니다.

서버 업로드가 실패해요

프로젝트가 선택되어 있는지, 네트워크와 설정의 서버 주소가 올바른지 확인하세요. 파일 공유로 백업할 수도 있습니다.

파일은 무엇이 저장되나요?

기본적으로 방 메시 USDZ와 metadata JSON이 생성되며, 직접 촬영한 사진이 있으면 JPG 파일도 함께 포함됩니다.

이제 내 방을 3D로 만들어보세요

프로젝트를 준비하고 방의 벽과 바닥, 가구가 모두 보이도록 천천히 한 바퀴 비춰주세요.
